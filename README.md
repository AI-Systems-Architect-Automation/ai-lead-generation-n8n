
# AI Lead Generation System (n8n)

## Overview
This project demonstrates a fully automated lead generation system built with n8n.

The workflow collects leads, enriches data, and prepares structured output for outreach or CRM systems.

## Features
- Lead scraping (web/API sources)
- Data enrichment via APIs
- Email / contact extraction
- Structured output (Google Sheets / CRM)
- Automation with webhooks

## Tech Stack
- n8n
- APIs (HTTP Request)
- Webhooks
- JSON processing

## Workflow Structure
1. Trigger (Webhook / Schedule)
2. Data collection
3. Data enrichment
4. Filtering
5. Export to CRM / Sheets

## Use Cases
- B2B lead generation
- Sales automation
- Market research
